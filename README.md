# Weather-Days-of-the-Week

A bright and playful web game for preschool children to learn weather words and weekdays.

## How to run

1. Open `index.html` in a browser.
2. Or use a simple static server:
   - `npx http-server .`
   - Open the local address shown in the browser.

## Features

- Student login screen with 12 large buttons.
- Five auto-progressing learning levels.
- Weather and weekday recognition with speech.
- Sentence builder final task.
- Stars, leaderboard, and localStorage persistence.
